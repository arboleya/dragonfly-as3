BevelValues

public function BevelValues(distance:Number, angle:Number, highlightColor:Number, highlightAlpha:Number, shadowColor:Number, shadowAlpha:Number, blurX:Number, blurY:Number, strength:Number)

Create a BevelValues object.

Parameters

distanceThe offset distance of the bevel, in pixels (floating point). The default value is 4.
angleThe angle of the bevel, from 0 to 360 degrees. The default value is 45.
highlightColorThe highlight color of the bevel, 0xRRGGBB. The default value is 0xFFFFFF.
highlightAlphaThe alpha transparency value of the highlight color. Valid values are 0 to 1. For example, .25 sets a transparency value of 25%. The default value is 1.
shadowColorThe shadow color of the bevel, 0xRRGGBB. The default value is 0x000000.
shadowAlphaThe alpha transparency value of the shadow color. Valid values are 0 to 1. For example, 0.25 sets a transparency value of 25%. The default value is 1.
blurXThe amount of horizontal blur in pixels. Valid values are 0 to 255 (floating point). The default value is 4. Values that are a power of 2 (such as 2, 4, 8, 16 and 32) are optimized to render more quickly than other values.
blurYThe amount of vertical blur in pixels. Valid values are 0 to 255 (floating point). The default value is 4. Values that are a power of 2 (such as 2, 4, 8, 16 and 32) are optimized to render more quickly than other values.
strengthThe strength of the imprint or spread. The larger the value, the more color is imprinted and the stronger the contrast between the bevel and the background. Valid values are 0 to 255. The default value is 1.
