public function DropShadowValues(distance:Number, angle:Number, color:Number, alpha:Number, blurX:Number, blurY:Number, strength:Number)

Create a DropShadowValues object.

Parameters

distanceThe offset distance for the shadow, in pixels. The default value is 4 (floating point).
angleThe angle of the shadow, 0 to 360� (floating point). The default value is 45.
colorThe color of the shadow, in hexadecimal format 0xRRGGBB. The default value is 0x000000.
alphaThe alpha transparency value for the shadow color. Valid values are 0 to 1. For example, 0.25 sets a transparency value of 25%. The default value is 1.
blurXThe amount of horizontal blur. Valid values are 0 to 255 (floating point). The default value is 4. Values that are a power of 2 (such as 2, 4, 8, 16 and 32) are optimized to render more quickly than other values.
strengthThe strength of the imprint or spread. The higher the value, the more color is imprinted and the stronger the contrast between the shadow and the background. Valid values are 0 to 255. The default is 1.
