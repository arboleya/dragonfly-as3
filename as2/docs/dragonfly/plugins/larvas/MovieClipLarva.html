public function MovieClipLarva(target:Object, useFrames:Boolean, fps:Number)

Create a MovieClipLarva object.

Parameters

targetThe MovieClipLarva target.
useFramesIf true 'onEnterFreame' mode used, otherwise (false) 'setInterval' mode is used.
fpsIf useFrames is false, you can set the refresh interval for the 'setInterval' mode (default=30).
